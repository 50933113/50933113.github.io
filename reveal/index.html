硬體----CPU

CPU是什麼:

電腦CPU即是電腦的(中央處理器)，是英文Central Processing Unit的縮寫，CPU性能的好壞直接決定整個電腦的性能，也就是電腦中最核心的硬體之一。

作用:在電腦中負責一切的運算，也會有一個指揮的作用，例如:安排音效卡、網卡和顯卡在適當時候處理。

硬體------記憶體

電腦記憶體是甚麼:記憶體都是以晶片製造而成電腦記憶體或隨機存取記憶體（RAM）是系統的短期資料儲存區，存放電腦正在使用中的資訊，以便快速地存取

作用:電腦記憶體或隨機存取記憶體（RAM）是系統的短期資料儲存區，存放電腦正在使用中的資訊，以便快速地存取。

硬體------顯示卡

它主要負責把主機向顯示器發出的顯示訊號轉化為一般電器訊號，使得顯示器能明白個人電腦在讓它做什麼。顯示卡的主要晶片叫「顯示晶片」（Video chipset，圖形處理器或視覺處理器），是顯示卡的主要處理單元。顯示卡上也有和電腦記憶體相似的儲存器，稱為「顯示記憶體」，簡稱顯存。

硬體------硬碟

硬碟功能：儲存（文件、圖片、音樂、影片、程式……）

硬碟種類：HDD(傳統硬碟)、SSD(固態硬碟)

HDD（優點）：便宜、保存大量資料、耐用性高、功能性高

（缺點）：需要很多能量、容易過熱、不適合裝在筆記型電腦。

SDD（優點）：低功耗、無噪音讀取速度快

（缺點）：價格太高

軟體------應用程式

用於完成特定任務，而不僅僅是運行計算機系統。套用軟體可能由單個程式組成，例如影像檢視器：一小群程式（通常稱為軟體包），它們密切合作以完成任務，如電子表格或文本處理系統;更大的集合（通常稱為軟體套件）的相關，但獨立的程式和軟體包，具有一個共同的使用者介面或共用的數據格式

軟體------系統軟體

負責控制、集成和管理計算機系統的各個硬體元件，以便其他軟體和系統使用者將其視為一個功能單元，而不必關注低級別的細節，例如將數據從記憶體傳輸到磁碟，或將文本渲染到顯示幕上。而常見的作業系統有：Windows,ios,Linox

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.
